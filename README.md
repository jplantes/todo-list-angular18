# Levantar el proyecto


### Clonar le proyecto

```sh
git clone https://github.com/jplantes/todo-list-angular18.git
```

### Cambiar a la rama que corre JSON Server

```sh
git checkout with-json-server
```

### Iniciar JSON Server

```sh
npx json-server db\db.jso
```

### Iniciar Proyecto Angular

```sh
ng serve
```
